Installing HAPI TestPanel

Setting up Eclipse

Building TestPanel should be possible under any environment, but using Eclipse is recommended. First set up your environment:

  • Install Maven (3.0 or newer) and ensure it works from the command line.
  • Install the Windowbuilder plugin into Eclipse.

Check out the Project

Next, check out the HAPI root project.
