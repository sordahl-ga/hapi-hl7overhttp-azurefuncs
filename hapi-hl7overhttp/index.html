HL7 over HTTP

This is the homepage for the HL7 over HTTP initiative. Welcome!

HL7 over HTTP is a new transport mechanism aimed at replacing the traditional HL7 v2 Mininal Lower Layer Protocol (MLLP) transport with a modern, standards-based approach. We believe that MLLP should be retired completely, and we are working to make that a reality.

This site has two useful sections:

  • The Specification itself. We are soliciting opinions and comments on this specification, so please get in touch.
  • A Java based API for implementing HL7 over HTTP in your application. The Java based API can be embedded in applications that do use HAPI, and in applications that don't otherwise use HAPI. Our Java API is Apache licensed, so it is free for use in all kinds of software, including commercial software!
