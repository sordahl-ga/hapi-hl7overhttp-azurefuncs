Troubleshooting

 

This section provides a description of error codes.

 

Table 3          Command Line Troubleshooting

 

 

Problem

Probable Cause

Solution

File already exists error

There already is a generated file in the directory that was specified.

Either change the destination directory on the command line, or remove all previous generated files in the directory.

 

Invalid command line option error

An invalid flag was specified.

Type ConfGen h to get a list of all valid command line options.

 

Source path required error

The path to the XML profile was not specified.

Add the XML source path as the first argument on the command line.

 

Destination path required error

The path for the generated source was not specified.

Add a valid path where the generated source will be put.  Add the path as the second argument on the command line.

 

Package name required error

The name of the package was not specified.

Add a package name as the last argument on the command line.

 

Invalid destination path error

The destination path that was specified doesnt exist.

The destination path must be a valid system path, and it must exist.

 

Invalid source file error

The XML source file doesnt exist or isnt in the directory specified.

Make sure that the XML source file is in the specified directory.
