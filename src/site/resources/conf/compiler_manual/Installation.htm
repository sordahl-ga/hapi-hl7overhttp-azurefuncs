Installation

 

Perform the following steps to install the HAPI Conformance Classes System.

 

1.      Download the archive containing the HAPI Conformance Class System from the HAPI Web site at http://hl7api.sourceforge.net.

 

  1. Extract the contents of the archive to a local directory.

 

  1. Download the remaining software as listed under System Requirements.
